# LeCoSAI

Lens correction software using astronomical images

天体の位置を用いたレンズ補正ソフト

# 概要
カメラのレンズ歪みを補正するプログラム

星空の映像を使用する。

カメラで撮影したレンズの効果で歪んだ星空の星の位置と、
星のカタログから参照した本来の位置を比較し、
レンズの歪みの行列を計算する。
